# Clutch One

Projeto de e-commerce desenvolvido com PHP, MySQL, HTML, CSS e JavaScript.

## Acessar projeto online

https://clutchone.infinityfreeapp.com

## Funcionalidades

- catálogo de produtos;
- carrinho de compras;
- cálculo de frete;
- cupons de desconto;
- simulação de pagamento;
- painel administrativo;
- controle de pedidos e estoque.

## Tecnologias

PHP, MySQL, HTML, CSS e JavaScript.

## Área Administrativa

O projeto possui uma área administrativa completa para gerenciamento de produtos, pedidos, cupons e estoque.

Por motivos de segurança, o acesso ao painel administrativo é protegido por autenticação. Embora os arquivos do painel estejam presentes neste repositório para fins de avaliação e demonstração do desenvolvimento realizado, o acesso às funcionalidades administrativas exige credenciais válidas.

Essa medida foi adotada para evitar alterações indevidas no banco de dados da aplicação hospedada, garantindo a integridade das informações, dos produtos cadastrados e dos pedidos registrados.

Dessa forma, visitantes podem analisar livremente o código-fonte da área administrativa, mas não conseguem utilizar as funções de gerenciamento sem autorização.
